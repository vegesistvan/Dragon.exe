A Dragon genealógiai adatok nyilvántartására, ellenőrzésére, javítására és különböző leszármazotti és felmenői listák előállítására szolgáló Windows alatt futó program. Az adatbázist menuálisam GEDCOM fájlból és Szluha féle html fájlból tötthetjük fel. Az utóbbi a leginkább kidolgozott, legkevésbé a manuális bevitel.  
Töltsük le a Dragon.exe repositorit a Code->Download zip funkcoióval egy tetszőleges mappába. A dragon.exe fájlra kattintva elindul a program.
