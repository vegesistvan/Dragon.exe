A Dragon genealógiai adatok nyilvántartására, ellenőrzésére és különböző leszármazotti és felmenői listák készítésére szolgáló Windows alatt futó program.
Az adatbázis feltöltése történhet manuálisan, GEDCOM fájlból és Szluha féle htm fájlból. A legkidolgozottabb az utóbbi, a legkevésbé a manuális adatbevitel.
Töltsük le a Dragon.exe repositorit a Code->Dowload.zip funkcióval egy tetszőleges mappába, és bonsuk ki a Dragon.files.zip fájlt. A Dragon.exe fájlra kattintva a program elindul.
