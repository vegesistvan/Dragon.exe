A Dragon programot áthelyeztem a http://comfirm.hu/Dragon szerverre.
A program futtatásához szükséges fájlok ugyanúgy a Dragon.files.zip-ban vanna, mint ahogy a GitHub-on  voltak. A programot úgy tudjuk letölteni, hogy:
1.	A böngészőbe beírjuk http://comfirm.hu/Dragon  Ezzel beléptünk a szerverre.
2.	Bemegyünk a dragon mappába, ahol a letöltendő zip fájlok vannak.
3.	Rákattintunk a letöltendő fájlra, ezután megkérdezik, hogy hova akarjuk letölteni és milyen néven. Ha ezt megadjuk, a zip fájl máris a gépünkön van.
4.	A zip fájlt ki kell bontani egy tetszőleges mappába, ahonnan a Dragon.exe végrehajtható.

A Dragon egy genealógia adatok tárolásáre, ellenőrzésre, karbantartására és különböző genealógiai listák előállítására szolgáló program.
Az MS Windows alatt fut és az sqlite adatbáziskezelőt használja.
